# Pokemon The Card Game (TCG) w/ Vue JS (API Example)

## Description

This is nothing fancy, but it's designed as a simple Vue JS application to go along with a [blog post](https://intergalacticmeow.com/using-an-api-with-vue-js/) to teach folks about API usage with Axios & basic routes.

## Install
```
npm install
```

### Compiles and hot-reloads for Development
```
npm run serve
```

### Compiles and minifies for Production
```
npm run build
```
